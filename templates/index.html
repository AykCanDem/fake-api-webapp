GET request: request data from a specified resource

curl -X GET -H "Content-Type: application/json" https://fake-api-webapp.herokuapp.com/players/1

GET request: only returns response headers thanks to -I option

curl -X GET -I -H "Content-Type: application/json" https://fake-api-webapp.herokuapp.com

POST request: performs resource-specific processing on the payload

curl -X POST -H "Content-Type: application/json" https://fake-api-webapp.herokuapp.com/players \
-d '{"name" : "New_Player_Name","age": "New_Player_Age"}'

PUT request: replace the target resource with the request payload

curl -X PUT -H "Content-Type: application/json" https://fake-api-webapp.herokuapp.com/players/1 \
-d '{"player_id": 1, "name" : "Alex","age": "50"}'

DELETE request: deletes the specified resources

curl -X DELETE -H "Content-Type: application/json" https://fake-api-webapp.herokuapp.com/players/1
